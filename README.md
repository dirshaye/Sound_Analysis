En:
This project focuses on sound analysis using machine learning and speech processing. The main goals are to create audio visualizations, recognize speakers, perform speech-to-text transcription, and analyze emotions and topics in audio data. This project is divided into several user stories, which will be developed progressively.

Tr:
Bu proje, makine öğrenimi ve konuşma işleme kullanarak ses analizi yapmayı amaçlamaktadır. Temel hedefler, ses görselleştirmeleri oluşturmak, konuşmacıları tanımak, konuşmayı metne dönüştürme işlemi yapmak ve ses verilerinde duygu ve konu analizleri yapmaktır. Bu proje, aşamalı olarak geliştirilecek bir dizi kullanıcı hikayesine ayrılmıştır.
